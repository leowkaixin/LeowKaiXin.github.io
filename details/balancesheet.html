Balance Sheet Details

The Balance Sheet Storyboard displays details of the movements and porportion of each asset, liability and equity category. With the storyboard, the user is able to make comparsions with balance sheets itself and other subsidiaries over accounting period of choice, allowing the organisation to derive the most value from balance sheet and make more informed decisions. This storyboard consists of 5 dashboards:
1. Overview
2. Assets
3. Liabilities
4. Equity
5. Balance Sheet Analysis.

The first dashboard is the Overview Dashboard. The dashboard showcases a summary of the organisation finances, i.e. the orgainsation's financial position.

The second dashboard is the Assets Dashboard. Assets are divided into 2 main categories: - Current Assets and Fixed Assets.
The user is able to see the details of each card, i.e. cash and fixed deposits in different currency for Cash & Equivalent, aging graph for Account Receivables (AR)

The third dashboard is the Liabilities Dashboard. Liabilities are divided into 2 main categories: - Current Liabilities and Long-Term Liabilities. The user is able to see the details of each card, i.e. loans in different currency for Short Term (ST) Loans, aging graph for Account Payabes (AP)

The fourth dashboard is the Equity Dashboard. Equity are divided into 3 main categories:
1. Share Capital
2. Retained Earnings
3. Other Reserves

The final dashboard is the Balance Sheet Analysis Dashboard. This dashboard is divided into 2 main categories:
1. Top 8
2. All
In each category, the user is able to see the performance of 7 different metrics (Current Assets, Non-Current Assets, Total Assets, Current Liabilities, Non-Current Liabilities, Total Liabilities, Net Assets, Equity) across either the Top 8 countries or by the regions (Europe, Asia, USA) respectively.
